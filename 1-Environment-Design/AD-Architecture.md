# Active Directory Architecture

The Active Directory domain was designed to reflect a real enterprise structure.

## Organizational Units
- AuraxTech (Top-level)
- HR
- Development
- Finance
- IT
- Servers
- Workstations
- Groups
- Service Accounts

## Identity Design
- Departmental user accounts
- Role-based security groups
- Dedicated service accounts

## Administrative Model
- Separation of server and workstation administration
- Centralized management using Group Policy
- Kerberos-based authentication with service accounts
